🚀 Getting Started (Local Setup)

Follow these steps to run Divinora locally.

📌 Prerequisites

Ensure the following are installed on your system:

Node.js ≥ 18

npm ≥ 9 (comes with Node.js)

Git

Check versions:

node -v
npm -v
git --version

📦 Installation

Clone the repository:

git clone https://github.com/SUGAM-ARORA/divinora.git
cd divinora


Install dependencies:

npm install

⚙️ Environment Variables

Create a local environment file:

cp .env.example .env.local


If .env.example is not present, create .env.local manually.

Example:

NEXT_PUBLIC_APP_NAME=Divinora


ℹ️ If no environment variables are required, this step can be skipped.

▶️ Run the Development Server
npm run dev


Open your browser at:

http://localhost:3000

🏗️ Build for Production
npm run build
npm start

🛠️ Technology Stack

Framework: Next.js (App Router)

Styling: Tailwind CSS

UI Components: shadcn/ui

Icons: Lucide React

State Management: React Hooks + Context API

Backend / DB: Supabase

Authentication: Supabase Auth

AI Services: Hugging Face

Deployment: Vercel / Netlify
